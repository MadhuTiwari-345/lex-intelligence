### **3. Queue Layer (The Secret Sauce)**

**Tech Stack**:
- **Queue**: Redis + BullMQ (or Celery if Python-only)
- **Worker Pool**: Horizontal scaling based on queue depth

**Architecture Pattern**:

```typescript
// DON'T DO THIS (Synchronous):
app.post('/api/contracts/generate', async (req, res) => {
  const contract = await openai.generate(req.body); // Blocks for 30s
  res.json(contract);
});

// DO THIS (Asynchronous):
app.post('/api/contracts/generate', async (req, res) => {
  const jobId = await queue.add('generate-contract', req.body);
  res.json({ jobId, status: 'queued' });
});

// Worker Service (separate process):
queue.process('generate-contract', async (job) => {
  const contract = await openai.generate(job.data);
  await notifyClient(job.id, contract);
});
```

**Why This Matters**:
- If 100 users click "Generate" simultaneously, synchronous calls crash your server
- Queue handles spikes gracefully — workers process jobs one by one
- Main app stays responsive even during traffic spikes

**Scaling Strategy**:
```yaml
# Kubernetes-style scaling (or equivalent)
workers:
  min_replicas: 2
  max_replicas: 20
  scale_up_threshold: queue_depth > 50
  scale_down_threshold: queue_depth < 5
```

**Why pgvector Instead of Pinecone?**

| Factor | pgvector | Pinecone |
|--------|----------|----------|
| **MVP Complexity** | ✅ Single database | ❌ Two databases to manage |
| **Cost** | ✅ Included with Postgres | ❌ $70+/month starter |
| **Performance** | ✅ <10ms for <1M vectors | ✅ Similar at this scale |
| **Migration Path** | ✅ Easy to Pinecone later | ❌ Harder to migrate out |
| **Data Consistency** | ✅ ACID transactions | ❌ Eventual consistency |

**Migration Strategy**: Start with pgvector. If you hit 50M+ vectors or need auto-scaling, migrate to Pinecone.

```python
class ContractDraftingService:
    def __init__(self):
        self.llm = ChatOpenAI(model="gpt-4-turbo")
        self.templates = self._load_templates()
    
    async def generate_contract(self, request: ContractRequest):
        # 1. Select template based on contract type
        template = self.templates[request.contract_type]
        
        # 2. Apply jurisdiction-specific rules
        jurisdiction_rules = self._get_jurisdiction_rules(
            request.jurisdiction
        )
        
        # 3. Generate with RAG (retrieve similar contracts)
        similar_contracts = await self._retrieve_similar_contracts(
            request.description
        )
        
        # 4. Construct prompt with context
        prompt = self._build_prompt(
            template=template,
            user_input=request.description,
            jurisdiction=jurisdiction_rules,
            examples=similar_contracts
        )
        
        # 5. Generate contract
        contract = await self.llm.ainvoke(prompt)
        
        # 6. Post-process validation
        validated_contract = self._validate_contract(contract)
        
        return validated_contract
```

**Document Intelligence Pipeline**:

```python
class DocumentAnalysisService:
    async def analyze_document(self, document_url: str):
        # 1. Download document
        document = await self._download_document(document_url)
        
        # 2. OCR (if PDF/image)
        if document.type in ['pdf', 'image']:
            text = await self._ocr_document(document)
        else:
            text = document.text
        
        # 3. Clause extraction (NLP)
        clauses = await self._extract_clauses(text)
        
        # 4. Risk analysis (ML model)
        risks = await self._analyze_risks(clauses)
        
        # 5. Generate summary
        summary = await self._generate_summary(text, risks)
        
        return {
            'clauses': clauses,
            'risks': risks,
            'summary': summary,
            'risk_score': self._calculate_risk_score(risks)
        }
```

**RAG Pipeline for Case Law**:

```python
class CaseLawRAGService:
    async def search_case_law(self, query: str, jurisdiction: str):
        # 1. Generate query embedding
        query_embedding = await self._embed_query(query)
        
        # 2. Vector search in pgvector
        similar_cases = await self.db.query(
            """
            SELECT id, case_citation, content, metadata,
                   1 - (embedding <=> $1) as similarity
            FROM case_law_embeddings
            WHERE jurisdiction = $2
            ORDER BY embedding <=> $1
            LIMIT 10
            """,
            query_embedding, jurisdiction
        )
        
        # 3. Verify citations (CRITICAL for legal AI)
        verified_cases = await self._verify_citations(similar_cases)
        
        # 4. Rerank by relevance
        reranked_cases = await self._rerank_cases(query, verified_cases)
        
        # 5. Generate response with citations
        response = await self._generate_response(query, reranked_cases)
        
        return response
```

**Critical: Citation Verification**

```python
class CitationVerifier:
    """Prevent Mata v. Avianca-style hallucinations"""
    
    async def verify_citation(self, citation: str) -> bool:
        # 1. Parse citation
        parsed = self._parse_citation(citation)
        
        # 2. Query official legal databases
        # (e.g., Indian Kanoon, Casemine, Court websites)
        is_valid = await self._query_legal_database(parsed)
        
        # 3. Cross-reference with trusted sources
        if is_valid:
            await self._cache_verified_citation(citation)
        
        return is_valid
```

```python
class LegalRAGPipeline:
    def __init__(self):

        """

        """
        # Step 1: Generate query embedding
        query_embedding = await self.embedder.embed_query(query)

        similar_cases = await self.vector_db.similarity_search_with_score(
            query_embedding,
            k=10,

        verified_cases = []
        for case, score in similar_cases:
            is_valid = await self.citation_verifier.verify(case.citation)
            if is_valid:
                verified_cases.append(case)
        
        # Step 4: Rerank by relevance
        reranked_cases = await self._rerank_cases(query, verified_cases)
        
        # Step 5: Generate response with citations
        response = await self._generate_response(query, reranked_cases)
        
        return response
```

## **Layer 4: Citation Verification System**

### **The Hallucination Prevention Layer**

```python
class CitationVerifier:
    """
    Prevent Mata v. Avianca-style hallucinations
    """
    
    def __init__(self):
        self.official_databases = {
            'india': [
                'https://indiankanoon.org',
                'https://casemine.com',
                'https://supremecourtofindia.nic.in'
            ],
            'uk': [
                'https://www.bailii.org',
                'https://www.legislation.gov.uk'
            ],
            'us': [
                'https://www.courtlistener.com',
                'https://law.justia.com'
            ]
        }
    
    async def verify_citation(self, citation: str, jurisdiction: str) -> dict:
        """
        Verify a citation against official databases
        """
        # Step 1: Parse citation
        parsed = self._parse_citation(citation)
        
        # Step 2: Query official databases
        verification_results = []
        for db_url in self.official_databases[jurisdiction]:
            result = await self._query_database(db_url, parsed)
            verification_results.append(result)
        
        # Step 3: Cross-reference results
        is_valid = any(result['found'] for result in verification_results)
        
        # Step 4: Return verification status
        return {
            'citation': citation,
            'is_valid': is_valid,
            'confidence': self._calculate_confidence(verification_results),
            'sources': verification_results
        }
    
    async def verify_llm_output(self, text: str) -> dict:
        """
        Extract and verify all citations from LLM output
        """
        citations = self._extract_citations(text)
        
        verification_results = []
        for citation in citations:
            result = await self.verify_citation(citation)
            verification_results.append(result)
        
        # Flag any invalid citations
        invalid_citations = [
            r for r in verification_results 
            if not r['is_valid']
        ]
        
        return {
            'total_citations': len(citations),
            'valid_citations': len(citations) - len(invalid_citations),
            'invalid_citations': invalid_citations,
            'is_safe_to_use': len(invalid_citations) == 0
        }
```

```python
import hashlib
from functools import lru_cache

class ResponseCache:
    """
    Cache responses for repeated queries
    """
    
    @lru_cache(maxsize=10000)
    async def cached_llm_call(self, prompt_hash: str) -> str:
        """
        Only call API if not cached
        """
        # Check cache first
        if prompt_hash in self.cache:
            return self.cache[prompt_hash]
        
        # Call API
        response = await self.llm.ainvoke(prompt_hash)
        
        # Cache result
        self.cache[prompt_hash] = response
        
        return response
    
    def get_prompt_hash(self, prompt: str) -> str:
        """
        Generate hash for prompt
        """
        return hashlib.sha256(prompt.encode()).hexdigest()
```

**Cost Impact**: 20-50% savings depending on query repetition.

## **RAG Pipeline — Complete Architecture**

### **The RAG Pipeline Explained**

RAG (Retrieval-Augmented Generation) combines:
1. **Retrieval**: Finding relevant documents from your knowledge base
2. **Generation**: Using those documents to generate accurate responses

**Why RAG Matters for Legal AI**:
- Reduces hallucinations by 60-80%
- Provides verifiable sources
- Enables up-to-date information
- Supports jurisdiction-specific knowledge

### **Complete RAG Implementation**

```python
class LegalRAGPipeline:
    """
    Production-ready RAG pipeline for legal research
    """
    
    def __init__(self):
        # Components
        self.embedder = OpenAIEmbeddings(model="text-embedding-3-small")
        self.vector_db = PostgreSQLVectorStore()  # pgvector
        self.llm = ChatOpenAI(model="gpt-4-turbo")
        self.reranker = CohereRerank()  # Optional but recommended
        self.citation_verifier = CitationVerifier()
    
    async def search_case_law(self, query: str, jurisdiction: str, k: int = 10) -> dict:
        """
        Complete RAG pipeline for case law research
        """
        # === STAGE 1: QUERY EMBEDDING ===
        query_embedding = await self.embedder.embed_query(query)
        
        # === STAGE 2: RETRIEVAL ===
        # Initial vector search
        initial_results = await self.vector_db.similarity_search_with_score(
            query_embedding,
            k=k * 2,  # Retrieve more for reranking
            filter={
                "jurisdiction": jurisdiction,
                "is_good_law": True  # Filter out overruled cases
            }
        )
        
        # === STAGE 3: RERANKING (Optional but Recommended) ===
        # Rerank by relevance
        reranked_results = await self.reranker.rerank(
            query=query,
            documents=[doc for doc, score in initial_results],
            top_k=k
        )
        
        # === STAGE 4: CITATION VERIFICATION (CRITICAL!) ===
        verified_cases = []
        for case in reranked_results:
            verification = await self.citation_verifier.verify(case.citation)
            if verification['is_valid']:
                verified_cases.append({
                    'case': case,
                    'verification': verification
                })
        
        # === STAGE 5: CONTEXT ASSEMBLY ===
        context = self._assemble_context(verified_cases)
        
        # === STAGE 6: GENERATION ===
        response = await self._generate_response(query, context)
        
        return {
            'answer': response['answer'],
            'sources': verified_cases,
            'citations': response['citations'],
            'confidence': response['confidence']
        }
    
    def _assemble_context(self, verified_cases: list) -> str:
        """
        Assemble retrieved cases into context for LLM
        """
        context_parts = []
        
        for i, case_data in enumerate(verified_cases, 1):
            case = case_data['case']
            verification = case_data['verification']
            
            context_parts.append(f"""
<case_{i}>
<citation>{case.citation}</citation>
<court>{case.court}</court>
<date>{case.date}</date>
<summary>{case.summary}</summary>
<key_holding>{case.key_holding}</key_holding>
<verification_status>{verification['status']}</verification_status>
</case_{i}>
""")
        
        return "\n".join(context_parts)
    
    async def _generate_response(self, query: str, context: str) -> dict:
        """
        Generate response with proper citations
        """
        prompt = f"""
<role>
You are a legal research assistant specializing in Indian law.
You provide accurate, well-cited legal analysis based on retrieved case law.
</role>

<context>
{context}
</context>

<instructions>
1. Answer the user's question based ONLY on the provided case law
2. Cite every case you reference using proper citation format
3. If the provided cases don't contain sufficient information, say so
4. Do not hallucinate or invent case law
5. Provide the most recent and relevant precedents
6. Explain how each cited case applies to the question
7. Use the format: [Case Name] ([Citation])
</instructions>

<question>
{query}
</question>

<output_format>
## Analysis
[Your legal analysis]

## Relevant Cases
1. [Case Name] ([Citation])
   - Key holding: [Brief summary]
   - Relevance: [How it applies]

2. [Case Name] ([Citation])
   - Key holding: [Brief summary]
   - Relevance: [How it applies]

## Conclusion
[Your conclusion based on the cases]
</output_format>
"""
        
        response = await self.llm.ainvoke(prompt)
        
        # Extract citations from response
        citations = self._extract_citations(response.content)
        
        return {
            'answer': response.content,
            'citations': citations,
            'confidence': self._calculate_confidence(citations, context)
        }
```

### **The Hallucination Problem**

**Why AI Hallucinates**:
- LLMs predict text, not facts
- They generate plausible-sounding but incorrect information
- Legal citations are particularly dangerous because they sound authoritative

**Real-World Impact**:
- Over 1,174 court cases worldwide involved AI hallucinations (as of April 2026)
- Lawyers have been sanctioned, fined, and disbarred

```python
class HallucinationPreventionSystem:
    """
    Complete system to prevent AI hallucinations in legal work
    """
    
    def __init__(self):
        self.citation_verifier = CitationVerifier()
        self.fact_checker = FactChecker()
        self.jurisdiction_validator = JurisdictionValidator()
        self.hallucination_detector = HallucinationDetector()
    
    async def validate_legal_output(self, output: str, jurisdiction: str) -> dict:
        """
        Complete validation pipeline for legal AI output
        """
        validation_results = {
            'is_safe': True,
            'issues': [],
            'warnings': [],
            'verified_citations': [],
            'suspicious_content': []
        }
        
        # === LAYER 1: CITATION VERIFICATION ===
        citations = self._extract_citations(output)
        
        for citation in citations:
            verification = await self.citation_verifier.verify(
                citation, 
                jurisdiction
            )
            
            if verification['is_valid']:
                validation_results['verified_citations'].append(citation)
            else:
                validation_results['is_safe'] = False
                validation_results['issues'].append({
                    'type': 'invalid_citation',
                    'citation': citation,
                    'reason': verification['reason']
                })
        
        # === LAYER 2: FACT CHECKING ===
        facts = self._extract_factual_claims(output)
        
        for fact in facts:
            fact_check = await self.fact_checker.verify(fact, jurisdiction)
            
            if not fact_check['is_verified']:
                validation_results['warnings'].append({
                    'type': 'unverified_fact',
                    'fact': fact,
                    'confidence': fact_check['confidence']
                })
        
        # === LAYER 3: JURISDICTION VALIDATION ===
        jurisdiction_issues = await self.jurisdiction_validator.validate(
            output, 
            jurisdiction
        )
        
        if jurisdiction_issues:
            validation_results['issues'].extend(jurisdiction_issues)
            validation_results['is_safe'] = False
        
        # === LAYER 4: HALLUCINATION DETECTION ===
        hallucination_score = await self.hallucination_detector.detect(output)
        
        if hallucination_score > 0.7:  # High risk threshold
            validation_results['is_safe'] = False
            validation_results['issues'].append({
                'type': 'potential_hallucination',
                'score': hallucination_score,
                'reason': 'High probability of hallucinated content'
            })
        
        return validation_results
```

```python
class CitationVerifier:
    """
    Verify legal citations against official databases
    """
    
    def __init__(self):
        self.official_databases = {
            'india': {
                'indiankanoon': 'https://indiankanoon.org',
                'casemine': 'https://casemine.com',
                'supreme_court': 'https://supremecourtofindia.nic.in'
            },
            'uk': {
                'bailii': 'https://www.bailii.org',
                'legislation': 'https://www.legislation.gov.uk'
            },
            'us': {
                'courtlistener': 'https://www.courtlistener.com',
                'law_cornell': 'https://law.cornell.edu'
            }
        }
        
        self.cache = {}  # Cache verified citations
    
    async def verify(self, citation: str, jurisdiction: str) -> dict:
        """
        Verify a citation against official databases
        """
        # Check cache first
        if citation in self.cache:
            return self.cache[citation]
        
        # Parse citation
        parsed = self._parse_citation(citation)
        
        if not parsed:
            return {
                'is_valid': False,
                'reason': 'Invalid citation format'
            }
        
        # Query official databases
        verification_results = []
        
        for db_name, db_url in self.official_databases[jurisdiction].items():
            result = await self._query_database(db_url, parsed)
            verification_results.append(result)
        
        # Cross-reference results
        is_valid = any(result['found'] for result in verification_results)
        
        # Calculate confidence
        confidence = self._calculate_confidence(verification_results)
        
        # Cache result
        verification = {
            'is_valid': is_valid,
            'confidence': confidence,
            'sources': verification_results,
            'parsed': parsed
        }
        
        self.cache[citation] = verification
        
        return verification
    
    def _parse_citation(self, citation: str) -> dict:
        """
        Parse legal citation into components
        """
        # Example: "Mohan Lal v. State of Rajasthan, (2023) 5 SCC 123"
        import re
        
        pattern = r'(.+?)\s+v\.\s+(.+?),\s*\((\d{4})\)\s*(\d+)\s*(\w+)\s*(\d+)'

        if match:
            return {
                'plaintiff': match.group(1).strip(),
                'defendant': match.group(2).strip(),
                'year': match.group(3),
                'volume': match.group(4),
                'reporter': match.group(5),
                'page': match.group(6)
            }
        
        return None
```

### **Hallucination Detection Patterns**

```python
class HallucinationDetector:
    """
    Detect potential hallucinations in AI output
    """
    
    def __init__(self):
        self.suspicious_patterns = [
            # Fake case names
            r'(Mercy v\. Mankind|Justice v\. Fairness|Truth v\. Lies)',
            
            # Unrealistic ratios
            r'\d+:\d+:\d+ ratio',  # Like 100:1:1
            
            # Generic placeholders
            r'\[Case Name\]|\[Citation\]|\[Court\]',
            
            # Unrealistic dates
            r'(18\d{2}|19\d{2})\s+SCC',  # Old cases in modern format
            
            # Suspiciously perfect citations
            r'\(\d{4}\)\s+1\s+\w+\s+1'  # First case of first volume
        ]
        
        self.legal_terms = self._load_legal_terms()
    
    async def detect(self, text: str) -> float:
        """
        Detect probability of hallucination (0-1)
        """
        hallucination_score = 0.0
        
        # Check for suspicious patterns
        for pattern in self.suspicious_patterns:
            matches = re.findall(pattern, text)
            hallucination_score += len(matches) * 0.2
        
        # Check for unrealistic legal terminology
        words = text.split()
        suspicious_words = [
            word for word in words 
            if word not in self.legal_terms and len(word) > 10
        ]
        hallucination_score += len(suspicious_words) * 0.05
        
        # Check citation density
        citations = self._extract_citations(text)
        if len(citations) == 0 and 'case' in text.lower():
            hallucination_score += 0.3  # Mentions cases but no citations
        
        # Normalize score
        return min(hallucination_score, 1.0)
```
